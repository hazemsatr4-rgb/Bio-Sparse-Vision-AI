# Bio-Sparse Vision AI 🧠

An ultra-fast Neural Network engine for autonomous systems, developed by **karar salooo**.

### Features:
- **High Performance**: Optimized for mobile processors.
- **Sparsity-Driven**: Uses only 10% of neural paths for decision making.
- **Persistent Memory**: Self-learning capabilities with `.npy` storage.

### How to use:
1. Clone the repo.
2. Run `python bio_sparse_ai.py`.
